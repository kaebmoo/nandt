# คู่มือการใช้งานระบบนัดหมายออนไลน์ NudDee (นัดดี)

> สำหรับผู้ดูแลระบบของโรงพยาบาล/คลินิก (Hospital Admin)
> อัปเดตล่าสุด: มิถุนายน 2026

---

## 1. ภาพรวมระบบ

NudDee ประกอบด้วย 2 ส่วนที่ผู้ใช้เกี่ยวข้อง:

| ส่วน | ผู้ใช้ | เข้าถึงผ่าน |
|---|---|---|
| **Dashboard จัดการนัดหมาย** | เจ้าหน้าที่โรงพยาบาล | `/dashboard` (ต้อง login) |
| **หน้าจองสาธารณะ** | ผู้รับบริการ/คนไข้ | `/book` (ไม่ต้อง login) |

แต่ละโรงพยาบาลมีพื้นที่ของตัวเองแยกขาดจากกัน ระบุผ่าน subdomain:
- โหมดพัฒนา: `http://localhost:5001/dashboard?subdomain=ชื่อโรงพยาบาล`
- โหมดใช้งานจริง: `https://ชื่อโรงพยาบาล.nuddee.com/dashboard`

## 2. การเริ่มต้นใช้งาน

### 2.1 การได้บัญชีมี 2 ทาง

1. **สมัครเองผ่านหน้าเว็บหลัก** — กรอกชื่อโรงพยาบาล, subdomain ที่ต้องการ, ข้อมูลผู้ดูแล แล้วระบบสร้างทุกอย่างให้ทันที
2. **ทีมงาน NudDee สร้างให้** (ผ่าน Super Admin) — จะได้รับอีเมลและรหัสผ่านจากทีมงาน

### 2.2 ระบบเตรียมอะไรให้บ้างโดยอัตโนมัติ

ทันทีที่สร้างโรงพยาบาลใหม่ ระบบจะตั้งค่าเริ่มต้นให้พร้อมเปิดรับจองทันที:

| รายการ | ค่าเริ่มต้น |
|---|---|
| ตารางเวลาทำการ | "เวลาทำการเริ่มต้น" จันทร์-ศุกร์ 08:30-16:30 |
| ประเภทการนัดหมาย | "นัดหมายทั่วไป" (30 นาที) และ "ปรึกษา/ติดตามอาการ" (15 นาที) |
| เจ้าหน้าที่ผู้ให้บริการ | "เจ้าหน้าที่ให้บริการ" 1 คน ผูกกับตารางเวลาแล้ว |

> **สำคัญ:** ค่าเริ่มต้นทั้งหมด**แก้ไขหรือลบได้**จากเมนูตั้งค่า ระบบตั้งให้เพื่อให้เริ่มใช้ได้เลย
> ไม่ต้องเข้าใจโครงสร้างระบบก่อน — เมื่อพร้อมค่อยปรับให้ตรงกับการทำงานจริงของหน่วยงาน

### 2.3 Checklist บน Dashboard

เมื่อ login ครั้งแรก Dashboard จะแสดงการ์ดสถานะ:
- ถ้าตั้งค่าครบ → แสดงลิงก์หน้าจองสาธารณะพร้อมปุ่ม "คัดลอกลิงก์" สำหรับนำไปแชร์
- ถ้ายังไม่ครบ (เช่น ลบค่าเริ่มต้นออก) → แสดง checklist 3 ขั้นตอนพร้อมปุ่มลัดไปตั้งค่าแต่ละเรื่อง

## 3. โครงสร้างแนวคิดของระบบ (อ่าน 2 นาที เข้าใจทั้งระบบ)

สามสิ่งนี้ทำงานร่วมกัน:

```
ตารางเวลาทำการ (Availability Template)
   "เปิดรับนัดวันไหน เวลาอะไรบ้าง"  เช่น จ-ศ 08:30-16:30
        │
        ├── เจ้าหน้าที่ (Provider) — ใครเป็นคนให้บริการในตารางนี้
        │      ผูกเจ้าหน้าที่เข้ากับตารางเวลา ระบบจะจ่ายคิวให้อัตโนมัติ
        │
        └── ประเภทการนัดหมาย (Event Type) — บริการอะไร ใช้เวลากี่นาที
               เช่น "ตรวจสุขภาพ 30 นาที" ผูกกับตารางเวลา เพื่อรู้ว่าจองช่วงไหนได้
```

ผู้รับบริการเห็นเฉพาะ **ประเภทการนัดหมาย** → เลือกวันจากปฏิทิน (ระบบเปิดเฉพาะวันตามตารางเวลา หักวันหยุด) → เลือกเวลา → กรอกข้อมูล → รับอีเมลยืนยัน

## 4. การตั้งค่าแต่ละส่วน

### 4.1 เวลาทำการ (เมนู ตั้งค่า → 🕐 ตั้งค่าเวลาทำการ)

- แก้ไขวันและช่วงเวลาของตาราง "เวลาทำการเริ่มต้น" ได้โดยตรง
- สร้างตารางเพิ่มได้หลายชุด เช่น "คลินิกนอกเวลา เสาร์ 09:00-12:00"
- ค่า "จำนวนคิวพร้อมกันสูงสุด" (max concurrent slots) กำหนดว่าช่วงเวลาเดียวกันรับซ้อนกันได้กี่คิว
- กำหนดวันปิดเฉพาะกิจ (Date Override) ได้ เช่น ปิดอบรมครึ่งวัน

### 4.2 เจ้าหน้าที่ผู้ให้บริการ (เมนู ตั้งค่า → 👥 จัดการผู้ให้บริการ)

- เปลี่ยนชื่อ "เจ้าหน้าที่ให้บริการ" เป็นชื่อจริง หรือเพิ่มแพทย์/เจ้าหน้าที่เพิ่มเติม
- แต่ละคนต้อง**ผูกกับตารางเวลาทำการ**อย่างน้อย 1 ตาราง จึงจะรับคิวได้
- ปิดการใช้งานชั่วคราว (is_active) ได้โดยไม่ต้องลบข้อมูล

### 4.3 ประเภทการนัดหมาย (เมนู ตั้งค่า → 📅 ประเภทการนัดหมาย)

ตัวเลือกสำคัญของแต่ละประเภทนัด:

| ตัวเลือก | ความหมาย |
|---|---|
| ระยะเวลา (นาที) | ความยาวของหนึ่งคิว |
| จองล่วงหน้าอย่างน้อย (ชม.) | กันการจองกระชั้นชิด (ค่าเริ่มต้น 4 ชม.) |
| จองล่วงหน้าได้ไม่เกิน (วัน) | เปิดปฏิทินให้จองล่วงหน้ากี่วัน (ค่าเริ่มต้น 60 วัน) |
| จำนวนจองสูงสุดต่อวัน | จำกัดโควต้ารายวัน (เว้นว่าง = ไม่จำกัด) |
| Buffer ก่อน/หลัง | เวลาพักระหว่างคิว |

### 4.4 วันหยุด (เมนู ตั้งค่า → 📅 จัดการวันหยุด)

- **หน่วยงานใหม่ได้วันหยุดราชการไทยของปีปัจจุบันอัตโนมัติทันทีตอนสร้าง** (จากข้อมูลธนาคารแห่งประเทศไทย)
- ระบบ sync วันหยุดราชการปีใหม่ให้อัตโนมัติปีละครั้ง (ต้องเปิด Celery Beat — ดูคู่มือติดตั้ง)
  หรือกด sync เองได้จากหน้าจัดการวันหยุด
- เพิ่มวันหยุดเฉพาะของหน่วยงาน แก้ไข หรือปิดวันหยุดที่ไม่ใช้ได้เองทั้งหมด
- วันหยุดจะถูกปิดในปฏิทินหน้าจองทันที

## 5. หน้าจองสาธารณะ

- ลิงก์อยู่ที่การ์ดบน Dashboard (ปุ่ม "คัดลอกลิงก์") หรือเมนู 🔗 หน้าจองสาธารณะ
- ผู้รับบริการไม่ต้องสมัครสมาชิก ใช้แค่ชื่อ + อีเมลหรือเบอร์โทร
- หลังจองสำเร็จจะได้**รหัสนัดหมาย** (เช่น `VN-A1B2C3`) และอีเมลยืนยันอัตโนมัติ
- ผู้รับบริการใช้รหัสนี้ค้นหานัด เลื่อนนัด หรือยกเลิกได้เองจากหน้า "ค้นหานัดหมาย"
- กติกาที่ระบบบังคับ: ยกเลิก/เลื่อนได้ก่อนถึงเวลานัดอย่างน้อย 4 ชั่วโมง

### อีเมลอัตโนมัติที่ระบบส่งถึงผู้จอง

| เหตุการณ์ | อีเมล |
|---|---|
| จองสำเร็จ | ยืนยันการนัดหมาย + รายละเอียด + รหัสนัดหมาย |
| เลื่อนนัด | แจ้งวันเวลาใหม่ |
| ยกเลิกนัด | ยืนยันการยกเลิก |
| กู้คืนนัดที่ยกเลิก | ยืนยันการนัดหมายอีกครั้ง |

(ต้องตั้งค่า `MAIL_USERNAME`/`MAIL_PASSWORD` ในระบบก่อน — ดูคู่มือติดตั้งใน README)

## 6. การจัดการนัดหมายประจำวัน (Dashboard)

- **สรุปตัวเลข**: นัดวันนี้ / กำลังจะมาถึง / ทั้งหมด
- **แท็บรายการนัด**: กำลังจะมาถึง / ผ่านมาแล้ว / ยกเลิก จัดกลุ่มตามประเภทนัด
- กดที่นัดหมายเพื่อดูรายละเอียด เลื่อนนัด หรือยกเลิกแทนผู้รับบริการได้
- ปุ่ม "เพิ่มนัดหมาย" เปิดหน้าจองเพื่อบันทึกนัดแบบ walk-in ให้คนไข้

### เมื่อเจ้าหน้าที่เลื่อนหรือยกเลิกนัด ระบบแจ้งผู้รับบริการอย่างไร

| ข้อมูลติดต่อของผู้จอง | สิ่งที่ระบบทำ |
|---|---|
| มีอีเมล | ส่งอีเมลแจ้งอัตโนมัติ และแสดงข้อความยืนยันบนจอ |
| มีแต่เบอร์โทร | **แสดงข้อความเตือนสีเหลืองให้เจ้าหน้าที่โทรแจ้ง** พร้อมชื่อและเบอร์ของผู้จอง — กรุณาโทรทันทีหลังทำรายการ |
| ไม่มีทั้งคู่ | เตือนว่าไม่มีช่องทางติดต่อ ให้แจ้งผู้รับบริการผ่านช่องทางอื่นเอง |

> สำหรับผู้ดูแลระบบ: เปิดส่ง SMS อัตโนมัติแทนการโทรได้ด้วยการตั้งค่า
> `ENABLE_SMS_NOTIFICATIONS=true` (ต้องมีบัญชี NT SMS) — ค่าเริ่มต้นปิดไว้

## 7. สำหรับทีมงาน NudDee (Super Admin)

เข้าผ่าน Admin Panel (port 5002 — เปิดเฉพาะภายใน):

1. **สร้างโรงพยาบาลใหม่**: Tenants → Create — ระบบสร้าง schema + ข้อมูลเริ่มต้นให้ครบอัตโนมัติ
   (เห็นข้อความยืนยันว่า seed สำเร็จใน flash message)
2. **สร้างบัญชี admin ให้โรงพยาบาล**: ในหน้า tenant → Create Admin User
3. **เปิด/ปิดการรับจองสาธารณะ** หรือระงับ tenant ชั่วคราวได้จากหน้าเดียวกัน
4. การลบเป็น soft delete — กู้คืนได้

สร้างบัญชี Super Admin คนแรกด้วย: `python scripts/create_super_admin.py`

## 8. คำถามพบบ่อย (FAQ)

**Q: หน้าจองสาธารณะไม่แสดงช่วงเวลาให้เลือกเลย**
ตรวจตามลำดับ: (1) ประเภทนัดเปิดใช้งานอยู่ไหม (2) ประเภทนัดผูกกับตารางเวลาแล้วหรือยัง (3) ตารางเวลามีวัน/เวลาอย่างน้อย 1 ช่วง (4) มีเจ้าหน้าที่ active ที่ผูกกับตารางนั้น (5) วันที่เลือกไม่ใช่วันหยุด — หรือดู checklist บน Dashboard ซึ่งสรุปให้อัตโนมัติ

**Q: ลบค่าเริ่มต้นที่ระบบสร้างให้ได้ไหม**
ได้ทั้งหมด แต่ต้องมีตารางเวลา + เจ้าหน้าที่ + ประเภทนัดของตัวเองแทน มิฉะนั้นหน้าจองจะไม่มีอะไรให้จอง (Dashboard จะเตือนผ่าน checklist)

**Q: ผู้จองไม่ได้รับอีเมลยืนยัน**
ตรวจว่าระบบตั้งค่า SMTP แล้ว (`MAIL_USERNAME`, `MAIL_PASSWORD`) และดู log ของ FastAPI — บรรทัด `📧 ส่งอีเมล ... สำเร็จ` หรือ `❌ ส่งอีเมล ... ล้มเหลว` จะบอกสาเหตุ การส่งอีเมลล้มเหลวไม่ทำให้การจองล้มเหลว (การจองยังถูกบันทึก)

**Q: เปลี่ยนเวลาทำการแล้ว นัดเดิมที่จองไว้นอกเวลาใหม่จะหายไหม**
ไม่หาย — นัดที่จองแล้วยังอยู่ตามเดิม การเปลี่ยนตารางมีผลเฉพาะการจองใหม่

**Q: ลืมรหัสผ่าน**
กดลิงก์ "ลืมรหัสผ่าน?" ที่หน้า login → กรอกอีเมล → รับรหัส OTP 6 หลักทางอีเมล (อายุ 15 นาที)
→ ตั้งรหัสผ่านใหม่ หากไม่ได้รับอีเมล ติดต่อทีมงาน NudDee ให้ reset ผ่าน Super Admin Panel
